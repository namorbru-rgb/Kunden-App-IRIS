# Kunden APP IRIS

Interaktiver UI-Prototyp für eine hochwertige Energie-Management-Plattform (IRIS).

## Inhalt
- `index.html` — vollständiger klickbarer Frontend-Prototyp
- `iris-prototype.png` — verifizierter Screenshot

## Fokus
- Energieflussvisualisierung
- PV / Batterie / Netz / LEG / vZEV
- Zeitreise durch Last- und Energieprofile
- Geräte- und Teilnehmerdetails
- Responsive Desktop-/Mobile-Ansicht
- reine UI mit Platzhalterdaten, ohne Backend

## Lokal starten
```bash
python3 -m http.server 8765
```
Dann öffnen:
```text
http://127.0.0.1:8765
```
